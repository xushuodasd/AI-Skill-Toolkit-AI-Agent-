# AI-Skill-Toolkit 🧰

> **A Comprehensive Collection of Production-Ready AI Agent Skills**

AI-Skill-Toolkit is a curated repository of modular, high-performance skills designed for modern AI agents and autonomous development environments (such as Trae, Claude Code, and Cursor). This toolkit provides a structured way to extend the capabilities of AI models across various professional domains.

---

## 🚀 Overview

This repository aims to bridge the gap between raw LLM capabilities and practical, task-specific execution. Each skill is designed with modularity and efficiency in mind, ensuring they can be easily integrated into different agentic workflows.

---

## 📂 Repository Structure

The toolkit is organized into several key domains, each containing specialized skills:

### 🛠️ Software Engineering
- **[Programming Enhancement](skills/programming-enhancement)**: Advanced hooks and polyglot tools for AI-driven development.
- **[Code Audit](skills/code-audit)**: Automated quality auditing and security analysis patterns.
- **[Zig Language](skills/zig-lang)**: Specialized support and references for Zig programming.

### 💼 Professional Productivity
- **[Bidding Automation](skills/bidding-automation)**: Tools for AI-assisted proposal and tender document generation.
- **[Office Processing](skills/office-processing)**: Programmatic manipulation of PDF, Word, and Excel documents.
- **[Product Planning](skills/product-planning)**: High-level product strategy and feature orchestration skills.

### 🎨 Creative & UI/UX
- **[UI Design](skills/ui-design)**: Design system management and automated UI component generation.
- **[Video Remotion](skills/video-remotion)**: Programmatic video creation framework using React.
- **[Copywriting](skills/copywriting)**: Guidelines for clear, effective technical writing and AI content refinement.

### 🧠 Intelligence & Research
- **[QA Architect](skills/qa-architect)**: Structuring complex queries and brainstorming workflows.
- **[Search](skills/search)**: Enhanced discovery and information retrieval mechanisms.
- **[Creator](skills/creator)**: Development kit for benchmarking and packaging new skills.
