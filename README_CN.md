# AI-Skill-Toolkit 🧰

> **生产级 AI Agent 技能库与工具集**

AI-Skill-Toolkit 是一个精心策划的模块化、高性能技能仓库，旨在为现代 AI Agent 和自动化开发环境（如 Trae, Claude Code, Cursor）提供功能扩展。该工具集为增强 AI 模型在各个专业领域的实际执行能力提供了一套结构化的方案。

---

## 🚀 项目概述

本项目致力于在 LLM（大语言模型）的原始能力与实际任务执行之间架起桥梁。每个技能在设计时都充分考虑了模块化和执行效率，确保它们能够轻松集成到不同的智能体（Agentic）工作流中。

---

## 📂 仓库结构

本工具集按专业领域划分为以下核心模块：

### 🛠️ 软件工程
- **[编程增强](skills/programming-enhancement)**: 用于 AI 驱动开发的高级钩子和多语言支持工具。
- **[代码审计](skills/code-audit)**: 自动化的代码质量审计与安全分析模式。
- **[Zig 语言](skills/zig-lang)**: 为 Zig 编程语言提供的专项支持与参考资料。

### 💼 专业生产力
- **[标书自动化](skills/bidding-automation)**: AI 辅助的提案与标书文件生成工具。
- **[办公文档处理](skills/office-processing)**: 对 PDF, Word, Excel 等文档的编程式操作。
- **[产品规划](skills/product-planning)**: 高阶产品策略分析与功能编排技能。

### 🎨 创意与 UI/UX
- **[UI 设计](skills/ui-design)**: 设计系统管理与自动化的 UI 组件生成。
- **[视频 Remotion](skills/video-remotion)**: 使用 React 构建的编程式视频创作框架。
- **[文案写作](skills/copywriting)**: 提供简洁、高效的技术写作指南与 AI 内容优化规范。

### 🧠 智能与研究
- **[问答架构师](skills/qa-architect)**: 用于结构化复杂查询与头脑风暴的工作流。
- **[搜索增强](skills/search)**: 高效的信息发现与检索机制。
- **[技能创建器](skills/creator)**: 用于新技能基准测试与打包的开发套件。

---

## 🛠 快速开始

在本地环境探索或使用这些技能：

```bash
git clone https://github.com/xushuodasd/AI-Skill-Toolkit-AI-Agent-.git
cd AI-Skill-Toolkit-AI-Agent-
```

每个技能文件夹中都包含一个 `SKILL.md` 文件，详细记录了该技能的用途、参数及使用示例。

---

## 🛡️ 设计原则

1. **模块化**: 技能高度独立，易于插件化集成。
2. **高效性**: 为 Token 使用效率和执行速度进行深度优化。
3. **专业性**: 聚焦于真实世界的生产级任务。
